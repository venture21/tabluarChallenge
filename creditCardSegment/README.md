# 신용카드 고객 세그먼트 분류 AI 경진대회

DACON 공식 경진대회: https://dacon.io/competitions/official/236460/data

## 대회 개요

신용카드 사용 데이터를 기반으로 고객을 5개의 세그먼트(A, B, C, D, E)로 분류하는 다중 분류 문제입니다.

## 제출 형식

```csv
ID,Segment
TEST_00000,A
TEST_00001,B
TEST_00002,C
...
```

- **ID**: 고객 고유 ID (TEST_00000 ~ TEST_99999)
- **Segment**: 예측된 세그먼트 (A, B, C, D, E 중 하나)
- 제출 파일 크기: 100,000개 행

## 평가 지표

Macro F1-Score

## 참고 사항

### 기간 표기법
- **B0M**: 당월 (Base 0 Month)
- **B1M**: 1개월 전
- **R3M**: 최근 3개월간 (Recent 3 Months)
- **R6M**: 최근 6개월간
- **R12M**: 최근 12개월간

### 용어 설명
- **신판**: 신용카드 판매 (일시불 + 할부)
- **CA**: 현금서비스 (Cash Advance)
- **CL**: 카드론 (Card Loan)
- **RV**: 리볼빙 (Revolving)
- **RP**: 정기결제 (Recurring Payment)
- **BL**: 블랙리스트 (Black List)

### 데이터 특성
1. **시계열 데이터**: 6개월치 월별 데이터가 제공됨
2. **클래스 불균형**: A, B 클래스가 매우 희소함
3. **다중 테이블**: 8개의 테이블을 ID와 기준년월로 조인하여 사용
4. **파일 형식**: Parquet (효율적인 컬럼 기반 저장 형식)
